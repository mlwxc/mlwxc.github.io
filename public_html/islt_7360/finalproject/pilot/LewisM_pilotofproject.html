Mission statement

This website is creating a list of genres for fiction and nonfiction that people can explore. The user will then be able to explore the books associated with this genre. When they find a book they would like to know more about it will link to a separate page. On this page, the author, a summary of the book, and a link to a website the user would be able to buy it.
